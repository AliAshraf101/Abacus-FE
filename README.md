# Banking Homepage - Next.js App

This Next.js app combines all sections from Part1 through Part9 into a single homepage.

## Sections Included (in order):

1. **Part1**: Navigation, Hero, IndustrySolutions
2. **Part2**: StatsSection, ProductsSection, OnPremisesSection
3. **Part4/Part5**: Enterprise Data Sources
4. **Part6/Part7**: Knowledge Management
5. **Part8**: Multi-Platform
6. **Part9**: Truth Assurance / Abacus Intelligence

All styling, colors, design, and layout remain exactly the same as the original sections.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the homepage.
